Like every single project I have ever completed, I always seem to run into some issues. During this Python Text Adventure project, I encountered two issues but did manage to solve them. First, I seemed to keep repeating the same mistake in my choice inpout statement. Everytime I ran the program, the output was jumbled up as not all of the words fit into what I needed to print. Basically, half of what needed to print was being printed. To solve this, I just added another print function, separate from the choice-input function so that there was a enough space for all the words, and left inside the parenthesis of the choice-input function blank. Second, I would forget to put some of my functions in the state machine, so when the output just kept repeating, but this was rather easy to solve as I simply jsut put the functions that were missing back intro the state machine.
